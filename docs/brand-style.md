# Lens — Brand Style

## Capitalization

**"Lens" is always capitalized** when referring to the product, brand, or any
product-derived noun. This applies in:

- Prose, headlines, body copy, slide content
- Product UI copy and labels
- Output documents (PDFs, briefs, lens documents)
- Marketing materials, decks, one-pagers
- README files, public-facing docs

### Examples

- ✅ Lens
- ✅ Lens Project
- ✅ Lens document, Lens output, Role Lens
- ✅ "I'm building Lens"
- ❌ lens, lens project, lens document

### Lowercase exceptions

Lowercase is preserved only in:
- File paths and directories: `zelman/lens/`, `lens-app-five.vercel.app`
- Code identifiers: `lensSchema`, `parseLens()`, `lens.ts`
- URLs and routing paths: `/api/lens`, query parameters
- Existing third-party content (don't alter quoted material)

### First use

In any document longer than a single screen, define on first use:
> *Lens — the AI-guided identity discovery and bidirectional matching product.*
